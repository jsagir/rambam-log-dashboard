# Log Files

## How to Add Logs

Place your Rambam log files (`.txt` or `.json` format) in this folder.

The dashboard will automatically load and analyze all log files when it starts.

## File Format

Log files should be in newline-delimited JSON format with interaction data.

Example filenames:
- `20260222.txt`
- `20260223.txt`
- `20260224.json`

## After Adding Files

1. Push changes to GitHub
2. Render will automatically redeploy
3. Dashboard will show updated data

## Current Status

This folder is ready to receive your log files.
